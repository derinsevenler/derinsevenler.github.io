Derin Sevenler

Derin is a PhD student in Biomedical Engineering at Boston University, in the Optical Characterization & Nanophotonics Lab. His research is focused on designing new molecular diagnostics methods using principles from nano-photonics, and applying those methods to address today's challenges in cancer and infectious disease.

Links to online software

IRIS Processing is a MATLAB utility for making measurements with IRIS, a label-free biosensing technology. It's based on this paper. If you're interested in IRIS Processing or wave optics in general, you might also be interested in this interactive implementation of the reflectance of thin films.

Device Adapters for Micro-Manager

Micro-manager is an open-source software toolkit for controlling scientific microscopes. The below hardware adapters aren't part of the core micro-manager project but they are all BSD licensed, so feel free to use & improve them if they'll help your microscopy application.

  • GenICam: a driver for USB3Vision and GigEVision cameras that comply to the Generic Interface for Cameras.
  • ThorlabsKinesisStage: a driver for Thorlabs linear postioners using the Kinesis software interface.
  • XYPolluxStage: a driver for using two SMC-Pollux linear positioners as a single 2-axis device. It uses the Venus-2 protocol from Physik Instrumente (PI) over serial.
